# Semantic Textual Similarity
- Sentence BERT: https://github.com/KimRass/transformer_based_models/tree/main/sentence_bert

# Install Libraries
## For MacOS
- `pip install -r requirements_macos.txt`
## For Windows (64-bit)
- `requirements_windows.txt`의 첫 번째 줄 `annoy @ file:///annoy-1.17.0-cp37-cp37m-win_amd64.whl`을 `annoy @ file:///<whl 파일의 절대 경로>`와 같은 형태로 바꿔 주어야 합니다.
- 이때 whl 파일은 Python version에 따라 다른 것이 필요합니다. 예를 들어, Python 3.7.x의 경우 'annoy-1.17.0-cp37-cp37m-win_amd64.whl'을, Python 3.8.x의 경우 'annoy-1.17.0-cp38-cp38-win_amd64.whl'을 사용합니다. (https://www.lfd.uci.edu/~gohlke/pythonlibs/#annoy)
